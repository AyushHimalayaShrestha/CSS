Cascading Style Sheet which is used to design the web pages.

There are 3 types of Css. They are:
        1. Inline/Embedded Css ( Worst Practice)
        2. Internal Css
        3. External Css ( Best practice)
    
There are four types of selector.They are:
        1. Element Selector (direct name)
        2. ID selector (you have to use # symbol then only id name)
        3. Class selector (you have to use . symbol then only class name)
        4. Grouping selector
